The Xylus Toolkit extends functionality to Xylus Themes, providing custom post types and more.

== Description ==

The Xylus Toolkit provides support for portfolio items, testimonials, services, team members and more to supported Xylus themes. Simply install the plugin and the theme will determine which features to activate.

== Installation ==
This plugin can be installed directly from your site.

1. Log in and navigate to Plugins & Add New.
2. Type "Xylus Toolkit" into the Search input and click the "Search" button.
3. Locate the Xylus Toolkit in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.

It can also be installed manually.

1. Download the "Xylus Toolkit" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Xylus Toolkit" in the list and click the "Activate" link.



==Frequently Asked Questions==

= What is this plugin and why do I need it? =

The Xylus Toolkit provides extra functionality to the collection of Xylus Themes. The Toolkit adds a few custom post types. The plugin is not a requirement to use Xylus themes, but it will extend the themes to function as you see them in the demos.

= Where are the Toolkit settings? =

All options have been automated, and you do not need to change any settings. Features are activated based on their support defined in the theme.

= Can I use this plugin with other themes? =

The Xylus Toolkit was developed to extend the functionality of Xylus Themes specifically, however parts of it may be useful to other themes. If using your own theme, you may have to provide some extra styling or modifications to customize it to your needs.
